# GitHub Pages CV (WangRui-debug)

This folder contains:
- `index.html` — simple homepage that previews and links your CV PDF
- `cv.pdf` — your CV (renamed)

## How to publish (GitHub Pages)

1) Create a new repository named **WangRui-debug.github.io**
   - It must be exactly this name (GitHub Pages rule).

2) Upload these two files to the repository root:
   - `index.html`
   - `cv.pdf`

3) Enable Pages:
   - Repo **Settings** → **Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** (or master) / folder: **root**
   - Save

4) Your site:
   - https://wangrui-debug.github.io
   - Direct PDF: https://wangrui-debug.github.io/cv.pdf

## Optional: use a custom domain
If you later buy a domain, add a `CNAME` file with the domain name and configure DNS.
